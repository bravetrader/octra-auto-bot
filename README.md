## octra-wallet-gen
### Auto Tx bot
## Setup Guide

1. Clone the this repo
```
git clone https://github.com/bravetrader/octra-wallet-gen.git
```

```
cd octra-wallet-gen

```
3. activate venv  
```
python3 -m venv venv
```
```
source venv/bin/activate
```

```
pip3 install -r requirements.txt
```

4. Run the bot
```
python3 bot.py
```
Enjoy!
